Projeto Hand2Note Análise de Mãos de Poker

Repositório feito para a disciplina de Princípios de Desenvolvimento Web. Esse projeto busca consumir os arquivos gerados pelo software Hand2Note para disponibilizar ferramentas de análise de mãos de poker.

Principais Funcionalidades:

  • Construção de banco de dados a partir de arquivos.
  • Criação de gráficos interativos para ajudar análises.
  • Possibilidade de fazer anotações sobre as mãos e filtrar mãos de acordo com padrões predefinidos.
